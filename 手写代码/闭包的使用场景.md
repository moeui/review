## 闭包的使用场景

闭包的定义可以简要的用一个函数调用另外一个函数内的变量，变量不会被销毁来概括。下面来看下闭包的使用场景：

    const Storage = function () {
        const data = []
        this.set = (key, value)=>{
            data[key] = value
        }
        this.get = (key)=>{
            return data[key]
        }
    }

    const storage = new Storage()
    storage.set('test', 1)
    storage.get('test') // 1

其中，变量不会因为new Storage()完成之后就消失，还是留在内存中，当调用set、get方法还能获取data变量
